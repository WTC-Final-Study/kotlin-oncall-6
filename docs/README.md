## 기능 목록

### 비상 근무일 배정 규칙

- 연도는 고려하지 않으며, 매년 2월은 28일까지만 있다고 가정
- 비상 근무자는 평일, 휴일 순번에 각각 1회 편성되어야 한다.
- 연속 2일 근무 불가능
    - 순번상 특정 근무자가 연속 2일 근무하게 되는 상황에는, 다음 근무자와 순서를 바꿔 편성한다.
- 순서를 바꿔야하는 경우, 앞의 날짜부터 순서를 변경해야 한다.

### 휴일

- 주말(토, 일)
- 법정 공휴일
    - 1/1: 신정, 3/1: 삼일절, 5/5: 어린이날, 6/6: 현충일, 8/15: 광복절, 10/3: 개천절, 10/9: 한글날, 12/25: 성탄절

### 비상 근무 배정 월 및 시작 요일 입력

- 월과 시작 요일 정보를 입력받는다.

    ```
    비상 근무를 배정할 월과 시작 요일을 입력하세요> 5,월
    ```

    - 월, 요일을 쉼표(`,`)로 구분하여 입력

**예외 처리**

- 아래 경우에 해당하는 경우 다시 입력받는다.
    - 형식이 올바르지 않은 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`
    - 입력한 월이 1~12의 정수가 아닌 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`
    - 입력한 시작 요일이 일~토가 아닌 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`

### 비상 근무 순서 입력

- 평일 및 휴일 비상 근무 순서를 입력받는다.

    ```
    평일 비상 근무 순번대로 사원 닉네임을 입력하세요> 준팍,도밥,고니,수아,루루,글로,솔로스타,우코,슬링키,참새,도리
    휴일 비상 근무 순번대로 사원 닉네임을 입력하세요> 수아,루루,글로,솔로스타,우코,슬링키,참새,도리,준팍,도밥,고니
    ```

    - 각 이름을 쉼표로 구분하여 입력

**예외 처리**

- 아래 경우에 해당하는 경우 평일부터 다시 입력받는다.
    - 이름이 5자가 넘는 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`
    - 중복된 이름이 있는 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`
    - 이름이 5개 미만 또는 35개 초과 경우: `[ERROR] 유효하지 않은 입력 값입니다. 다시 입력해 주세요.`

### 비상 근무표 출력

- 평일이면서 법정 공휴일인 경우, 요일 뒤에 `(휴일)`을 표기한다.

```
5월 1일 월 준팍
5월 2일 화 도밥
5월 3일 수 고니
5월 4일 목 수아
5월 5일 금(휴일) 루루
5월 6일 토 수아
5월 7일 일 글로
5월 8일 월 루루
...
```
