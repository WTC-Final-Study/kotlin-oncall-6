# 구현 기능 목록

## 1. 값 입력 받기 / 검증 
- [x] 월, 시작요일 입력 받기
  - [x] 입력 형식 검증 (월,시작요일)
  - [x] 월 (1 ~ 12) 범위 확인
  - [x] 시작요일 (월 ~ 일) 범위 확인
  - [x] 시작 정보 저장
- [x] 평일 비상근무 순번 입력 받기
- [x] 휴일 비상근무 순번 입력 받기
- [x] 비상근무 순번 검증
  - [x] 닉네임 중복 여부 확인
  - [x] 닉네임 최대 5자 확인
  - [x] 순번 리스트 범위 (5명 이상, 35명 이하) 확인
  - [x] 순번 객체 저장

## 2. 달력 만들기
- [x] 입력된 월, 시작 요일에 맞게 객체 생성
- [x] 생성된 객체 리스트에 저장 Map<Integer, Day>

## 3. 비상근무 지정하기
- [ ] 평일, 휴일 확인하고 순번에서 차례대로 지정
- [ ] 전날 근무자인지 확인하고, 근무자일 경우 뒷 사람과 순서 바꾸기

## 4. 결과 출력하기
- [ ] 1일부터 마지막날까지 결과 출력
  - 평일이면서 법정공휴일일 경우에만 (휴일) 표기
