# 구현 기능 목록

## 1.값 입력 받기
- [x] 배정 월, 요일 입력 받기
- [x] 월, 요일 파싱
- [ ] 평일 비상 근무 순번 입력 받기
- [ ] 주말 비상 근무 순번 입력 받기

## 2. 값 검증
- [x] 배정 월,요일 입력 형식 검증
- [x] 월 (1~12) 범위 검증
- [x] 요일 (월~일) 범위 검증
- [ ] 순번 닉네임 1~5자 검증
- [ ] 순번 최소 5명 ~ 최대 35명 검증 

## 3. 달력 생성
- [ ] 입력된 월, 요일 기준으로 달력 생성

## 4. 스케줄 생성
- [ ] 순번에 맞게 평일, 주말 스케줄 생성
- [ ] 전날 근무한 사람과 연속될 경우 뒷 순번과 바꾸기

## 5. 출력
- [ ] 스케줄 출력
